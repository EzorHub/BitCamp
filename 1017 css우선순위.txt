우선순위를 고려하셔서 적용하시면 됩니다.
간단히 말씀드리면
.table {} 에 적용된 스타일을 덮어쓰고 싶다면, 간단히 2가지 방법이 있습니다.

1. 덮어쓸 css 를 기존 적용된 부분 뒤에 정의합니다.
.table {margin: 10px}
.table {margin: 20px}
대충 이런식으로 정의하면 뒤에게 적용됩니다.
이런식으로
<link ..... bootstrap.min.css>
<link ..... common.css> 이렇게 하시고 덮어 쓸 부분을 common.css 에 정의 하시면 됩니다.

2. 셀렉터의 우선순위를 높이세요. important 는 꼭 필요할때만 사용하시길 추천합니다.
이것도 간단히 적으면
table.table { margin: 10px }
.table { margin: 20px }
이렇게 하면 10px 이 먹습니다.
셀렉터에 우선순위가 있습니다. 제가 여기 적기 보다는 직접 검색해 보시면 정리 잘된글 많을꺼에요.
